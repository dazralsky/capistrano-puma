# Capistrano::Puma

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano-puma', github: "seuros/capistrano-puma"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install capistrano-puma

## Usage

    # Capfile

        require 'capistrano/puma'
